# Prerequisites Installation Guide

## Prerequisites

Before getting started, ensure you have the following installed:

1. **[Node.js](https://nodejs.org/)** (includes `npm`)
2. **Package Manager** – You can use any of the following:
   - [`pnpm`](https://pnpm.io/) – _Recommended!_
     Install it globally:

     ```bash
     npm install -g pnpm
     ```

   - [`yarn`](https://classic.yarnpkg.com/lang/en/docs/install/)
   - `npm` (comes with Node.js)

3. **Modern Web Browser**
   Any latest version of [Chrome](https://www.google.com/chrome/),
   [Firefox](https://www.mozilla.org/firefox/), etc.
4. **Python** (Optional – required only if using `pre-commit`)
   Recommended: Python 3.7 or higher
   - Install [pre-commit](https://pre-commit.com/#install):

     ```bash
     pip install pre-commit
     ```

> For more details on setting up Python, refer to the [Python Installation Guide](https://www.python.org/).
